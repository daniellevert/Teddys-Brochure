# Getting Started

### Install Prequisite

```
npm install --save
npm install --save nodemon -g
npm install --save concurrently -g
```

### Prequisite Extension For VsCode

```
ESLint
```

### How To Run Dev Mode

```
npm run dev
```

### How To Run Official Mode

```
npm start
```
